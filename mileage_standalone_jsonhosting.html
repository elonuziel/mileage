גיבוי ענן (JSONHosting.com)

האתר JSONHosting.com מאפשר אחסון JSON פשוט ללא הרשמה.

כדי ליצור גיבוי חדש לגמרי: השאר את השדות למטה ריקים ולחץ "שמור לענן". האפליקציה תיצור אחסון חדש ותשמור כאן אוטומטית את ה-ID ואת ה-Edit Key (מפתח עריכה).

⚠️ חובה עליך לשמור גיבוי של ה-ID ומפתח העריכה במקום בטוח! בלי הנתונים האלו הגיבוי שלך יאבד ולא תוכל לגשת אליו מטלפון אחר. כדי להתחבר לגיבוי קיים: הזן את ה-ID (או הקישור המלא) כדי לטעון נתונים. אם ברצונך גם לעדכן גיבוי קיים, חובה להזין את ה-Edit Key שלו.
