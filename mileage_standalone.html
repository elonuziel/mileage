גיבוי ענן (JSONBin.io)

האתר JSONBin.io מאפשר אחסון JSON בחינם. הכנס את X-Access-Key שלך, ולחץ "שמור לענן" כדי ליצור אוטומטית מקום אחסון (Bin) חדש לנתונים שלך, או הכנס גם Bin ID כדי לעדכן גיבוי קיים.
